selectedIntervallStorageTable constant

String const selectedIntervallStorageTable

Name of the table for storing time intervals to display.

It is used to store json representations of IntervallStorage objects. Data is saved as fields, to save space on the disk and because no changes in the data format are expected. The field names are made to match the variable names in the class.

Note that this table has 2 keys: profile_key to support multiple profiles and storage_id to support saving multiple entries per profile (e.g. export screen and main page).

Format: CREATE TABLE selectedIntervallStorage(profile_id INTEGER, storage_id INTEGER, stepSize INTEGER, start INTEGER, end INTEGER, PRIMARY KEY(profile_id, storage_id))
